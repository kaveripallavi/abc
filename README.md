# AI Resume Analyzer

An intelligent resume analysis platform that leverages AI to extract insights, match job descriptions, identify skill gaps, and provide actionable feedback to job seekers and recruiters.

## Overview

AI Resume Analyzer helps job seekers optimize their resumes and recruiters efficiently screen candidates using advanced AI-powered analysis, scoring, and recommendations.

## Key Features

- **Resume Parsing**: Extract structured data from resumes (PDF, DOCX, TXT)
- **AI-Powered Analysis**: Analyze resume quality, clarity, and effectiveness
- **Job Match Scoring**: Compare resumes against job descriptions
- **Skill Gap Analysis**: Identify missing skills for target roles
- **Personalized Recommendations**: Get actionable suggestions for improvement
- **Batch Processing**: Analyze multiple resumes efficiently
- **Comparison Reports**: Compare multiple resumes side-by-side
- **ATS Optimization**: Check resume compatibility with Applicant Tracking Systems

## Architecture

- **Backend**: Node.js + Express
- **Frontend**: React + TypeScript
- **Database**: PostgreSQL
- **AI Engine**: LLM integration (OpenAI/Claude)
- **File Processing**: PDF/DOCX parsing
- **Job Queue**: Bull for async processing
